How does javascript work?

প্রধানত জাভাস্ক্রিপ্ট একটি ক্লায়েন্ট সাইড স্ক্রিপ্টিং ভাষা। যেসব স্ক্রিপটিং ভাষা রয়েছে তাদের মধ্যে জাভাস্ক্রিপ্ট বহুল ব্যবহৃত হয় এবং জাভাস্ক্রিপ্টের সবচেয়ে দক্ষ স্ক্রিপ্টিং ল্যাঙ্গুয়েজ হিসেবে জানা যায়। জাভাস্ক্রিপ্ট ব্রাউজার এর ভিতরে রান করে। ব্রাউজার এর ভিতরে জাভাস্ক্রিপ্ট ইঞ্জিন থাকে যার মাধ্যমে জাভাস্ক্রিপ্ট কোড রাণ করে । বিভিন্ন ব্রাউজারের নিজস্ব জাভাস্ক্রিপ্ট ইঞ্জিন রয়েছে যেমন ফায়ারফক্সের Spidermonkey, ক্রোমের v8, microsoft-এর Chakra । নিচের ধাপগুলো দ্বারা জাভাস্ক্রিপ্ট রান করে -
1. The script gets loaded as a UTF-16 byte systeam from either the network , cache or a worker and passed to a byte stream decoder.
2. The byte stream decoder decodes the byte into tokens. The tokens are sent to the parser.
3. The parser generates nodes based on the tokens and creates an abstract syntax tree.
4. The interpreter walks through the AST and genertes byte code.
5. The byte code and type feedback are sent to the optimizing complier which generates highly optimized machine code.

Difference between local storage and session storage!!

#Local Storage

1.স্টোরেজঃ সাইজ 10 mb .
2. ডাটা নিজ হাতে ডিলিট করতে হয়।
3. কোন নির্দিষ্ট ট্যাবের জন্য কাজ না করে যে কোন ট্যাবের জন্য কাজ করে।
4. প্রতিটি HTTP অনুরোধের জন্য ডেটা সার্ভারে ফেরত পাঠানো হয় (html, images, javascript, CSS, ইত্যাদি) হ্রাস করা ক্লায়েন্ট এবং সার্ভারের মধ্যে ট্র্যাফিকের পরিমাণ

#Session Stroage

1. স্টোরেজঃ সাইজ 5 mb .
2. ব্রাউজার অথবা ট্যাগ বন্ধ করলে স্টরেস ক্লিয়ার হয়ে যায়।
3.ডেটা শুধুমাত্র window ভিতরে পাওয়া যায় যেখানে এটি সেট করা হয়েছিল
4. কোন নির্দিষ্ট ট্যাবের জন্য কাজ করে অথবা অ্যাপ বন্ধ করে ডাটা মুছে যায় ।
